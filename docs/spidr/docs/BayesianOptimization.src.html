15 | ||| This module defines functionality for Bayesian optimization, the data-efficient optimization of
16 | ||| objective functions. Bayesian optimization recommends new points at which to query your
17 | ||| objective by placing a probabilistic model over historic data then, typically, optimizing an
18 | ||| _acquisition function_ which quantifies how useful it would be to evaluate the objective at any
19 | ||| given set of points.

38 | ||| A Bayesian optimization loop as a (potentially infinite) stream of values. The values are
39 | ||| typically the observed data, and the models of that data. The loop iteratively finds new points
40 | ||| with the specified `tactic` then updates the values with these new points (assuming some
41 | ||| implicit objective function).
42 | |||
43 | ||| @tactic The tactic which which to recommend new points. This could be optimizing an acquisition
44 | |||   function, for example. Note this is a `Morphism`, not a function.
45 | ||| @observer A function which evaluates the optimization objective at the recommended points, then
46 | |||   updates the values (typically data and models).
47 | export
48 | loop :
49 |   (tactic : Reader i (Tensor shape dtype)) ->
50 |   (observer : Tensor shape dtype -> i -> i) ->
51 |   i ->
52 |   Stream i
53 | loop tactic observer = iterate (\ii => observer (runReader ii tactic) ii)
